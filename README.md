# 🏦 Loan Approval Prediction System

## 📌 Overview

This project predicts whether a loan should be **Approved or Rejected** using Machine Learning.

It analyzes applicant details such as income, credit score, and existing loans to make predictions.

---

## 🚀 Features

* Predict loan approval instantly
* Simple and user-friendly interface
* Uses Logistic Regression model
* Built with Streamlit

---

## 🛠️ Technologies Used

* Python
* Pandas
* Scikit-learn
* Streamlit

---

## ▶️ How to Run Locally

1. Install dependencies:

```
pip install -r requirements.txt
```

2. Run the app:

```
streamlit run app.py
```

---

## 🌐 Deployment

The app is deployed using Streamlit Cloud.

## 🎯 Output

* Displays whether loan is:

  * ✅ Approved
  * ❌ Rejected
